Background

This is part of the environment setup and is a prerequisite for the tutorials in this series. The pre-built virtual machine Database App Development VM contains Oracle Database and Oracle Application Express 18.1 installed.

What Do You Need?

  • Install the latest version of Oracle VM VirtualBox
  • Download and import the Database App Development VM in to the Oracle VM VirtualBox

Install the Oracle VM VirtualBox

To install the Oracle VM VirtualBox:

  1. Go to Oracle Application Express Hands on Lab page and click Oracle VM Virtual Box. The Downloads page opens.
  2. On the Downloads page, download the Oracle VM VirtualBox Base Packages installer, as applicable.
  3. Run the installer to install the Oracle VM VirtualBox.

Import the Database App Development VM

To import the Database App Development VM into the Oracle VM VirtualBox:

  1. Download the Database App Development VM from the Oracle Application Express Hands on Lab page.
    Note: Since this is a file of 8 GB, it will take some time for the download to complete.
  2. Once the download is complete, import the Database App Development VM (.ova file) into the Oracle VM VirtualBox. 
  3. Open the Oracle VM VirtualBox and under File, click Import Appliance.
  4. In the Import Virtual Appliance dialog, click and navigate to the location where you have downloaded the .ova file. Once you locate the file, the components and details of the VM display in the Import Virtual Appliance dialog, as shown in the screenshot. Click Import.
  5. Once the import completes, the VM displays in the left pane of the Oracle VM VirtualBox. Select the VM and click Start to run the Database App Development VM.
  6. Open the Database App Development VM in your Oracle VirtualBox and click Start.  
  7. Oracle Application Express 18.1 opens in a browser. Click APEX. 
  8. On the Oracle Application Express login page, enter the following details:
    •  Workspace: Enter obe
    • Username: Enter obe
    • Password: Enter oracle
    • Click Sign in.
